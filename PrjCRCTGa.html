Georgia CRCT Analysis

UPDATE 8/10/2016! The Milestones test has replaced the CRCT for use in Georgia. For my latest pivot using Milestones data, go to that page.

I finally had some time to update previous analysis I had done in the past for CRCT scores in Georgia. There have been some changes for the better! The data "dump" formatting has been somewhat standardized. Two additional columns were included starting 2011. Might be congressional district related. Not really of use to me from what I can tell. They also provided data up to 2013 so we have fairly recent data in government terms! The standard format greatly helped things get done faster this year. It is almost like somebody read my previous project report?

It has been reported CRCT testing will be replaced by another test in Georgia. Not sure if that will happen in 2015? Depending upon how the new testing results get reported, comparisons might become much more difficult. I'm not going to waste much time on the speculation until results are available.

I will also look at implementing weighting factors for the primary subjects of study. As of right now, it is similar analysis to previous one (focus on percentage exceeds scoring, all subjects). It is a pivot so feel free to configure however you like.

The SQL work is provided for those who want to see background into the work or recreate and verify the results.
